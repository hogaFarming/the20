深入理解Javascript作用域链和闭包

发表于 2017-09-21

很多编程语言都有代码块，在大部分编程语言中的代码块都有相应的“变量作用域”。

变量指在代码块中声明的变量，所谓变量作用域即可以访问该变量的“代码区域”，如Java：

{ 
    int a = 10;
} 
int b = a + 10; // error: a cannot be resolved to a variable

代码块内声明的变量，在代码块之外是不可见，即访问不到的，这就是“块级作用域”。
