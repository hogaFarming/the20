Sublime Text 3使用介绍

发表于 2017-09-21

开始记录些东西…

Sublime Text 是一款流行的代码编辑器，非常轻巧且功能强大，插件丰富，因此广受程序员尤其是前端青睐。

该软件售价79美刀，也可免费使用，所以请立刻移步官网下载：http://www.sublimetext.com/

本文目的，记录常用的功能、插件和设置等，以免下次重装系统后又得再找一遍 (☆_☆)/~~。

发现一个不错的社区，scoth.io

##常用功能

文件切换

Sublime text提供了快速切换文件的方法，快捷键为ctrl+p，在搜索框键入文件名即可，而且响应速度极快

页内跳转

如果当前文件相当大，比如一个包含大量方法的js文件，或者很大的css文件，可以使用该功能快速跳转，快捷键为ctrl+r

同时编辑

可以同时选中页面多处文本，同时编辑，这可以说是Sublime Text的特色功能，非常实用。有三种方法可以使用到这个功能：

  1. ctrl+d：同时选中当前和下一个相同文本

  2. ctrl+单击：将光标放到任意指定位置，非常炫酷

  3. ctrl+f：搜索并选中查找全部

代码块

Sublime Text提供了snippets代码块功能，比如编辑器自带的lorem文本，也可以创建自定义，或者使用插件提供的代码块，使用方法为lorem+tab键。这里有一篇介绍自定义snippets的文章 Working with Code Snippets in Sublime Text

其他快捷键

alt+shift+1-4：四窗口编辑

##插件安装

可以直接下载到Packages目录（菜单->preferences->packages）

也可以通过Package Control组件下载：

a. 按ctrl+` 调出console;

b. 执行以下命令并重启:

import urllib.request,os; pf = ‘Package Control.sublime-package’; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), ‘wb’).write(urllib.request.urlopen( ‘http://sublime.wbond.net/‘ + pf.replace(‘ ‘,’%20’)).read())

c. 按ctrl+shift+p，输入install加载插件搜索框

###常用插件

####Emmet

HTML/CSS代码自动补全功能，前端开发必用，其功能强大，将在后续文章补充记录

####Autoprefixer

为css3添加各浏览器私有前缀，兼容版本可自定义，插件数据来自caniuse

####Git

搜索Git插件安装，如果系统没有配置git环境变量，安装完成后在设置中添加git.exe路径

####Terminal

在当前文件或根文件夹打开命令行，快捷键分别为ctrl+alt+t和ctrl+shift+alt+t

####Alignment

代码对齐

####DocBlockr

方便为代码添加注释，输入/**+tab键

####其他

#####SideBarEnhancements

扩展边栏右键菜单

#####markdown高亮

推荐Markdown Extended，不过需要配合Monokai Extended使用
